Android-Hijacking

this is a process in which the attacker gets access to the victims personal information through thier smartphones. attackers get into phone systems just by following some steps such letting the target install their virus and getting enoh permissions to exploit victims privacy. such tools usesd by attackers to get into smartphones is called 'rat'(remote access toolkit) which can even pass through all the security measures accuired by the user

ahmyth

ahmyth is a very famous rat used by begginer pentesters. ahmyth provides you a framework for construction and establishment of connections between attacker and victim. it provides with basic information such live location, contacts, sms, camera activation, microphone activation and access storage. to make it work we first generate the payload[virus] and install it in victims device with all the permissions it asks. then we will put the framework to listen on the ip and port provided to the payload where it will connect. and now we can easily monitor victims activity
